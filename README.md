# Space Shooter Game

## Description
Space Shooter is an exciting arcade-style game built using Python and Pygame. Players control a spaceship, dodge enemies, and shoot to survive while progressing through increasingly challenging levels.

## Features
- Dynamic starfield background
- Multiple levels with increasing difficulty
- Player spaceship with shooting mechanics
- Enemy fighter jets with unique behaviors
- Explosions and particle effects
- Health system and score tracking

## Installation
1. Ensure Python 3.10+ is installed on your system.
2. Install the required dependencies:
   ```bash
   pip install pygame
   ```

## How to Play
1. Run the game:
   ```bash
   python code/main.py
   ```
2. Use the following controls to play:
   - **Arrow Keys / WASD**: Move the spaceship
   - **Spacebar**: Shoot bullets
3. Survive waves of enemies and progress through levels.

## Troubleshooting
- **Out of Memory Error**: Ensure your system has enough memory and avoid running other heavy applications simultaneously.
- **ModuleNotFoundError: No module named 'pygame'**: Install Pygame using the command `pip install pygame`.
